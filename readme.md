# instalace
Vyzaduje node, npm
```
git clone https://github.com/Sekky61/iis.git
cd iis
npm i
cd client
npm i
cd ..
```

## Development
`npm run dev` spusti client i server. Pokud chcete jen jedno, pak:

```
npm run server
npm run client
```
